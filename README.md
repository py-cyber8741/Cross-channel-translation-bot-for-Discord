# Cross-channel-translation-bot-for-Discord
Cross-channel translation bot for Discord

https://github.com/py-cyber8741/Cross-channel-translation-bot-for-Discord/releases/download/1.0.1/transbot.exe

DiscordのBOTトークンは必須ですが、geminiは任意なので必須ではありません。


取扱説明書
本アプリケーションは、133言語に対応したDiscord用の多機能翻訳Botを、直感的なGUIから一元管理するためのシステムです。複数サーバーのチャンネル間相互翻訳、ダイレクトメッセージ（DM）の管理、リアルタイム翻訳通知、そしてGemini APIを活用したAI推敲機能を備えています。

例えば、チャンネルAとチャンネルBをグループとし、チャンネルAで発言した内容をチャンネルBに指定した言語に翻訳して出力します。逆も同様です。


1. 初期設定と起動
Botを稼働させるための最初のステップです。
1. トークンの設定
   * 画面右上の [⚙️ 設定] ボタンをクリックします。
   * 「基本設定」タブの [Bot Token:] に、Discord Developer Portalから取得したBotのトークンを入力します。
   * （任意）/ai コマンドを使用する場合は、[Gemini API Key] も入力します。
   * [保存] ボタンを押して設定ウィンドウを閉じます。
2. Botの起動
   * 画面下部の中央にある [Bot起動] ボタンをクリックします。
   * 一番下のシステムログエリアに [System] ✅ Bot logged in as: ... と表示されれば起動成功です。
2. メイン画面の基本操作と翻訳グループ設定
相互翻訳を行いたいチャンネルを「グループ」として登録します。同じ「Grp（グループ名）」に属するチャンネル間でメッセージが自動翻訳・転送されます。
翻訳チャンネルの登録・更新
画面中央の入力フォームを使用します。
* GID (Guild ID): 対象サーバーのID。
* Grp (Group Name): 翻訳グループの名前（例: global-chat）。
* ChID (Channel ID): 対象チャンネルのID。
* Lan (Language): 翻訳先の言語コード（例: ja, en, zh-TW）。
* Webhook送信: チェックを入れると、Botが元の発言者の名前とアイコンを模倣（Webhook機能）して翻訳を送信します。
* 国旗スタンプ翻訳: チェックを入れると、そのチャンネル内の発言に対して国旗絵文字（🇯🇵や🇺🇸など）でリアクションした際、その言語に翻訳した結果をBotがリプライします。
【入力のヒント（サイドバーの活用）】
* [📡 チャンネル取得] ボタンを押すと右側にサーバーとチャンネルのツリーが表示されます。対象のチャンネルをダブルクリックすると、GIDとChIDが自動入力されます。
* [🌐 言語一覧] ボタンを押すと右側に言語リストが表示されます。使用したい言語をダブルクリックすると、Lanが自動入力されます。
* 入力が完了したら [追加/更新] ボタンを押すと、下の階層ツリーに登録されます。
3. 🔔 新着翻訳フィード（通知機能）
画面左上には、Botが参加している全サーバーの翻訳メッセージをリアルタイムで監視できるフィード画面があります。
* リアルタイム表示: 誰かが発言すると、自動的に指定した言語（デフォルトはUI言語）に翻訳され、時刻・サーバー名・チャンネル名・送信者と共にリストの最上部に追加されます。
* ログビューワーへの連携: リスト上の通知をダブルクリック（または右クリック ➔ 「📝 このチャンネルのログを表示」）すると、その発言があったチャンネルの「ログビューワー」が即座に開き、前後の会話の文脈を確認できます。
* 設定変更: 保持件数や通知自体のON/OFF、翻訳先言語は [⚙️ 設定] 内の「🔔 翻訳フィード設定」から変更可能です。
4. ✉️ DM（ダイレクトメッセージ）管理機能
ユーザーからBot宛てに送られたDMを確認し、GUIから直接返信できる機能です。
画面右上の [✉️ DM] ボタンから開きます。
* 会話の表示: 左側のリストからユーザーを選択すると、右側にそのユーザーとのチャットログが表示されます（履歴はローカルに自動保存されます）。
* 返信機能:
   * 「そのまま送信」または「翻訳して送信」を選択できます。
   * 「翻訳して送信」を選んだ場合、入力したテキスト（例：日本語）を選択した言語（例：英語）にBotが自動翻訳してから相手に送信します。
* ➕ ユーザー追加/同期:
   * [追加]: 左中段の入力欄にユーザーIDを入力して追加ボタンを押すと、そのユーザーとの過去のDM履歴をDiscordから取得してリストに追加します。
   * [🔄 過去のDMをDiscordから同期]: Botのサーバー上に残っている過去のDM履歴を一括で取得し、リストを復元します。
5. チャンネル管理ツール
メイン画面下部の階層ツリー、または右側の「チャンネル取得」ツリーから、チャンネルを右クリック（またはダブルクリック）することで各種ツールを呼び出せます。
5.1 💬 メッセージ送信機能 (Message Sender)
Botとして指定チャンネルに発言します。
* 翻訳送信: あなたの母国語で入力したメッセージを、対象チャンネルの言語に自動翻訳して送信できます。
* 直近の発言の削除: 「🗑️ 直近のBot発言を削除」ボタンで、誤って送信したBotのメッセージを直近のものから順に取り消せます。
5.2 📜 ログビューワー機能 (Log Viewer)
チャンネルの過去の会話履歴を取得し、まとめて翻訳・閲覧できます。
* 翻訳対象言語: ドロップダウンから言語を選び [🌐 Translate] を押すと、画面内の全ログが一括翻訳されます。
* ※翻訳の際、「そのチャンネルに設定されている言語（Lan）」をソース言語として明示的に指定して翻訳するため、中国語が日本語と誤判定されるなどのAPIの弱点を回避した高精度な翻訳が行われます。
5.3 🌐 多言語チャンネル自動生成 (Auto Clone)
右側の「チャンネル取得」ツリーからチャンネルを右クリックして呼び出します。
* 選択したベースとなるチャンネル（例：#chat）を複製し、指定した複数の言語用チャンネル（例：#chat-en, #chat-ko, #chat-zh-TW）を全自動でDiscord上に作成します。
* 同時に、それらを一つの「相互翻訳グループ」として本アプリに自動登録します。
6. Discord上でのスラッシュコマンド操作
Discordのチャット欄から直接Botを操作することも可能です。
* /chset [チャンネル数]: （管理者用）コマンドを実行したサーバー内で、現在閲覧できるチャンネルのリストを含むドロップダウンを表示します。選択した複数チャンネルを一括で相互翻訳グループとして設定します。
* /chdel: （管理者用）コマンドを実行したチャンネル、または指定したチャンネルが所属している翻訳グループを解除（削除）します。
* /ai [テキスト]: Gemini 2.5 Flash APIを使用し、入力したテキストを自然な文章に推敲・翻訳して送信します（※設定画面でのAPI Key登録が必要です）。
7. ⚙️ 設定画面の詳細 (Settings)
* UI Language: GUI全体の表示言語を即座に切り替えます。
* Log Count: 送信画面やログビューワーを開いた際に、過去何件のメッセージを取得するかを指定します。
* Spam Time Window / Spam Msg Limit:
指定した時間（分）内に、指定した回数（回）以上の連続投稿を行ったユーザーを「スパム」と判定し、自動的に翻訳機能を一時停止（ミュート）する機能です。
* BAN設定 / 一時停止リスト:
スパム判定されたユーザーは「一時停止リスト」に入ります。悪質な場合は右クリックから「BANリストに登録」することで、そのユーザーからのメッセージを完全に翻訳対象外にできます。特定のサーバー全体をBANすることも可能です。
